# Java Projects Collection 🚀

This repository contains multiple Java projects built to practice core programming concepts and real-world logic.

---

## 📌 Projects Included

### 🧮 1. Calculator
A basic calculator that performs arithmetic operations.

#### Features
- Addition
- Subtraction
- Multiplication
- Division
- User input handling

#### Run
javac Calculator.java  
java Calculator  

---

### 🏧 2. ATM System
A simple ATM simulation project.

#### Features
- PIN authentication
- Check balance
- Deposit money
- Withdraw money
- Menu-driven system

#### Run
javac ATM.java  
java ATM  

---

### 🔐 3. Login System
A basic login authentication system.

#### Features
- Username & Password validation
- 3 login attempts limit
- Account lock after failed attempts

#### Run
javac LoginSystem.java  
java LoginSystem
